# 🔱 Ashtavadhani (अष्टावधानी)

**Ashtavadhani** is an advanced Multi-Agent Collaborative Video Synthesis Engine. Inspired by the ancient Indian tradition of performing eight complex cognitive tasks simultaneously, this platform orchestrates **eight specialized AI agents** to edit raw footage, analyze audio beats, synchronize visual transitions, overlay sound effects, generate subtitles, and render final outputs.

---

## 🤖 The 8 Collaborative Agents

1. **Manager Agent** – Decides creative scope, edits layout, and assigns tasks.
2. **Vision Agent** – Explores clips, evaluates action intensity, and flags blur/exposure changes.
3. **Reference Agent** – Decodes edit pacing and patterns from target reference clips.
4. **Music Agent** – Profiles audio energy waves to synchronize cuts directly to beat drops.
5. **SFX Agent** – Pinpoints optimal highlights to overlay relevant sound effects (swooshes, hits).
6. **Transitions Agent** – Designs dynamic zooms, whips, and cut timing.
7. **Motion Graphics Agent** – Determines visual grades and text overlay branding styles.
8. **Caption Agent** – Automates speech-to-text transcription for custom kinetic subtitles.

---

## ⚡ Key Features

- **Multi-Agent Orchestration:** Dynamic JSON-based collaboration plan between agents.
- **Beat-Sync Technology:** Direct audio energy wave mapping using MoviePy & NumPy.
- **Dynamic Vibe Grading:** Automatically adapts color profiles (Moody Phonk, Cooking Warm, Tech Minimal).
- **Actual Quota Tracking:** Dynamic real-time calculation and visualization of remaining Google Gemini API calls.
- **Dual Engine System:** Automatically swaps from Gemini AI-enhanced processing to local heuristic pixel-analysis if API limits are reached.
- **Timeline Customization:** Live interactive audio/text overlay builder directly on the dashboard player.

---

## 🛠️ Technology Stack

- **Backend:** FastAPI (Python), MoviePy, NumPy, Whisper, Google Generative AI (Gemini)
- **Frontend:** Vanilla HTML5, CSS3 (Glassmorphism theme), JavaScript (ES6), Lucide Icons
